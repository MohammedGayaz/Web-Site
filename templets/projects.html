Projects:

I have developed several projects that showcase my technical skills and problem-solving abilities. These include:

Autonomous Robot

The project aims to create a robot capable of performing tasks and navigating a dynamic environment without human intervention. The robot is equipped with sensors and software using ROS2 to handle input, perform real-time control.

Face Identification and Attendance Management System

A face identification and attendance management system, created using Flask and Convolutional Neural Network, which can accurately identify students and mark attendance.

Diabetes Progression

I build a multivariate linear regression model from scratch using the sci-kit-learn library that predicts a quantitative measure of disease progression one year after baseline for diabetes patients.

Workout Tracker

Using the Nutritionix API, Sheety API, and Python I created a workout tracker that converts a natural language input into activity duration and calories burned, which are then stored in an Excel sheet using Sheety API.

Stock Price Alert

I built an application in Python that compares the stock prices of a stock form previous days and alerts the user of the percentage change in price along with the top 3 news articles of that company, utilizing the SMTPLIB and stock API.

Sudoku Solver

With the help of Back tracking algorithm, I wrote an program in Python that solve any Sudoku problem.

These projects demonstrate my proficiency in software development and my ability to apply technical knowledge to solve real-world problems.
